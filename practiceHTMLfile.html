Learning HTML Basics by example

HMTL is the main "programming language" for internet web pages. It is the base or foundationfor web pages. HTML is the first web programming language every programmer must learn. Following learning HTML you should also learn "CSS" or Cascading Style Sheets which another progamming language that is heavily used for styling web pages. This web page is designed to teach HTML by example.

HTML elements which are also called "tags" by some, follow some very simple rules and by example. HTML is NOT hard to learn in general.
Below are some of the basic rules:

  1. Almost every HTML tag must have a matching element tag.

  4. Almost every element in HTML is NOT "case-sensitive. This means you can type almost every HTML element in all lower-case letters.
  5. HTML is very "forgiving". This means you can mispell some tags but your web page will still load but with some errors. So double-check your code.
  6. Some of the most common HTML tags which do not have a matching element tag are:
    1. < BR > - Used for hard line breaks or like the "return" key on a keyboard.
    2. - This must be included to make all web pages HTML5 compliant. It tells the web browser what type of document the file is. Search the web for an explanation of what this tag does.
    3. < IMG > - Used for pictures and graphic files.
    4. < META > - Used to describe the information also called metadata within the webpage. This tag is placed in between the opening and closing Head tag
    5. < WBR > - Used for force hard line breaks in blocks of text when the when the text will not normally wrap around.
    6. This element is supported in most browsers.
  7. For information on HTML element visit the www.w3.org or World Wide Web Consortium or w3schools.com for free lessons on HTML, CSS and other web programming language basics.
